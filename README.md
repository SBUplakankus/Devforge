# Devforge

Devforge is now an Electron + Vite desktop application.

## Requirements

- Node.js 20+
- npm

## Install

```bash
npm install
```

## Run in development

```bash
npm run dev
```

This starts Vite for the renderer and launches Electron.

## Build renderer assets

```bash
npm run build
```

## Run Electron against the current project

```bash
npm run start
```
